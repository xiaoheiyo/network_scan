# 长Ping监控工具

基于 Python + tkinter 的批量 IP 长 Ping 监控工具，支持实时状态可视化、多状态告警、定时启停、日志归档、闪退重启等功能。

## 功能

- **多 IP 并发 Ping**：支持单 IP、CIDR（`192.168.1.0/24`）、范围（`192.168.1.1-255`）导入，支持 `IP,备注` 格式（逗号/制表符/分号分隔）
- **实时状态可视化**：7 种 Ping 状态以不同颜色格子展示，点击格子查看详情
- **参数实时调整**：Ping 间隔、超时、TTL、字节大小、显示格数，修改即时生效
- **备注管理**：右键设置备注，双击编辑 IP 和备注
- **禁用 IP**：右键禁用/启用指定 IP，禁用后带 `#` 前缀标记
- **定时启动/关闭**：支持跨夜调度（当日启动→次日关闭→再次启动）
- **告警声**：持续正常后出现异常时播放，可自定义各状态告警声（错误/警告/提示/蜂鸣）
- **日志记录**：每个 IP 独立日志文件，超过大小自动 gzip 压缩归档
- **配置持久化**：所有参数、颜色、备注、禁用状态自动保存到 `ping_config.json`
- **闪退自动重启**：启用后崩溃自动拉起（生成 `restart.bat`）
- **单文件打包**：使用 Nuitka onefile 模式打包，无需 Python 环境

## 使用方法

1. 下载 `gui_scan.exe` 直接运行
2. 导入 IP：点击「导入IP」选择 .txt/.csv 文件，或在底部输入框手动添加（按回车）
3. 点击「开始监控」启动 Ping
4. 右键 IP 可设置备注/禁用/删除
5. 通过「⚙ 设置」调整颜色、告警声、定时任务等

## 状态说明

| 状态 | 颜色 | 说明 |
|------|------|------|
| 正常连通 | 绿色 | 两次 Ping 均成功，延迟稳定 |
| 请求超时 | 红色 | 两次 Ping 均无响应 |
| 目标不可达 | 深红 | 目标不可达 |
| TTL传输中过期 | 黄色 | TTL 耗尽 |
| 找不到主机 | 紫色 | DNS 解析失败 |
| 部分丢包与延迟抖动 | 橙色 | 有丢包或延迟抖动 > 20ms |
| 仅首包超时 | 橙色 | 第一次超时，第二次成功 |

## 配置

所有配置保存在运行目录的 `ping_config.json` 中，包括：

- IP 列表与备注
- Ping 参数（间隔、超时、TTL、字节）
- 各状态颜色与告警声设置
- 定时任务与自动重启设置
- 禁用 IP 列表

## 构建

```bat
pip install nuitka
python -m nuitka --standalone --onefile --enable-plugin=tk-inter --windows-console-mode=disable --output-dir=. gui_scan.py
```

## 日志

- `log/<ip>.log`：每个 IP 的 Ping 记录
- `log/<ip>_YYYYMMDD_HHMMSS.log.gz`：超过大小限制后自动压缩归档
- `log/crash_*.log`：崩溃日志
- `error.log`：启动异常日志
